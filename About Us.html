About Me

Hello! I'm Mukudzei Chisenwa, a dedicated student with a strong interest in web development and front-end design. My goal is to become a professional web developer, and the Web Authoring class has played a big role in helping me move closer to that goal.

How This Class Helped Me

Taking this course introduced me to many essential web development skills and technologies. I learned how to create clean, well-structured HTML pages and apply CSS to style them beautifully. I also learned to write JavaScript to make my web pages interactive and dynamic.

One of the most valuable skills I learned was how to create responsive web designs. I now understand how to use media queries and flexible layouts so that websites look great on phones, tablets, and desktops. This is extremely important in today’s mobile-first world.

Another major takeaway was learning how to use the Bootstrap framework. Bootstrap helped me build responsive layouts much faster with its built-in classes for grids, buttons, forms, and components like navbars and cards. I used Bootstrap on several projects to ensure consistency and responsiveness without writing too much custom CSS.

Technologies and Techniques I Used

  • HTML5 – for page structure and semantic markup
  • CSS3 – for layout and styling using Flexbox and Grid
  • JavaScript – for interactive features and event handling
  • Bootstrap – to build responsive and professional-looking interfaces quickly
  • Responsive Design – using fluid grids, flexible images, and media queries
  • Accessibility – ensuring my content is usable by all users

This class gave me the foundation to build real-world websites and the confidence to keep learning and growing as a web developer. I’m excited to continue building my portfolio and eventually work in a role where I can use these skills every day.
